Runs fully in your browser — your documents are never uploaded anywhere. Your queue and extracted text are saved to this browser automatically, so closing the tab or losing connection won't lose your progress; hit "Clear" only when you actually want to wipe it. Large batches (100+ pages) take real time; keep this tab open while it works.
On handwriting: this engine is built and trained for printed text. Neat, well-spaced handwriting or block capitals can come through readable with "Enhance" mode on, but cursive or messy handwriting will be unreliable — that's a limit of the free on-device model, not a setting we can fully fix. For consistently accurate handwriting recognition you'd need a dedicated handwriting-OCR service (e.g. Google Cloud Vision or Azure's Read API), which requires a paid API key and a server call rather than running for free in the browser.
